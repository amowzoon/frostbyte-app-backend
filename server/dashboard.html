⬡ Radar
GET /api/radar/status ▶
Get radar initialization state, ports, and Redis connection.
POST /api/radar/initialize ▶
Initialize radar with automatic GPIO reset. Takes ~5–10s.
POST /api/radar/reset ▶
Hardware reset via GPIO17 (NRST pin). Reconfiguration needed after reset.
POST /api/radar/power_cycle ▶
Full USB power cycle (unbind → rebind → reinitialize). Use when radar is stuck with dead USB.
POST /api/capture [radar] ▶
Trigger a radar capture.
